# Examen Janvier 2025 - Password Checker

Ce projet contient une fonction de vérification de mot de passe développée en TDD.

## Règles de validation :
- Minimum 8 caractères
- Au moins un caractère spécial
- Au moins un chiffre
- Ne doit pas contenir "IPL" (insensible à la casse)
